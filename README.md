simbot
======

    Simbot is a 2 wheels differential drive robot simulator.

    It is implemented in C with OpenGL graphic.

* Prerequisites

    This project uses GLFW 3.0.4 to handle events and manage the OpenGL context.

* Build instructions

    - Build stripped binaries:
        make

    - Build with symbols:
        make debug
